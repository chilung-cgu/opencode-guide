## 🎯 API 開發概覽

OpenCode 可以幫助你快速構建健壯的 API 服務，從設計到測試，全程協助。

## 🚀 快速開始

### 🎯 **創建 Express API**

```bash
> 請創建一個 Express.js REST API：
  - TypeScript 支援
  - 使用者認證 (JWT)
  - 資料庫整合 (PostgreSQL)
  - 完整的錯誤處理
```

```bash
> 請創建一個 Python FastAPI 服務：

  - SQLAlchemy ORM
```

## 📐 API 設計

### 🎯 **RESTful 設計**

```bash
> 請設計一個電商系統的 RESTful API：
  - Users (用戶管理)
  - Products (商品)
  - Orders (訂單)
  - 遵循 REST 最佳實踐
```

AI 生成的設計：
```
📋 API 端點設計

用戶管理 /api/v1/users
├── POST   /users          創建用戶
├── GET    /users/:id      獲取用戶
├── PATCH  /users/:id      更新用戶
└── DELETE /users/:id      刪除用戶

商品管理 /api/v1/products
├── GET    /products       列出商品 (分頁)
├── POST   /products       創建商品
├── GET    /products/:id   獲取商品詳情
└── ...
```
