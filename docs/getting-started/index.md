## 🛠️ 前置需求

### 系統要求
- **作業系統**: Windows 10+, macOS 10.15+, 或 Linux (Ubuntu 18.04+)
- **記憶體**: 至少 4GB RAM（建議 8GB+）
- **網路**: 穩定的網際網路連接（用於 LLM API）
- **終端**: 支援 ANSI 轉義序列的終端模擬器

### 推薦的終端軟體
- **跨平台**: [WezTerm](https://wezterm.org), [Alacritty](https://alacritty.org)
- **macOS**: [iTerm2](https://iterm2.com), [Terminal.app](https://support.apple.com/guide/terminal/welcome/mac)
- **Linux**: [GNOME Terminal](https://wiki.gnome.org/Apps/Terminal), [Konsole](https://konsole.kde.org)
- **Windows**: [Windows Terminal](https://aka.ms/terminal), [PowerShell](https://docs.microsoft.com/en-us/powershell/)

### API 金鑰準備
準備至少一個 LLM 提供商的 API 金鑰：

#### 免費選項
- **OpenCode Zen** - OpenCode 團隊測試的模型（推薦新手）
- **Google Colab** - 有限的免費額度
- **Hugging Face** - 開源模型免費使用

#### 付費選項
- **OpenAI** - GPT-3.5/4 系列，$0.002/1K tokens 起
- **Anthropic** - Claude 系列，$0.015/1K tokens 起
- **Google Vertex AI** - Gemini 系列，$0.00025/1K tokens 起

## 🔧 故障排除準備

### 常見問題預覽
在開始之前，了解可能遇到的問題：

1. **終端不支援** - 某些終端可能不支援顏色或互動功能
2. **網路連線問題** - API 呼叫需要穩定的網路
3. **API 金鑰問題** - 金鑰格式、額度、權限等
4. **權限問題** - 某些系統可能需要管理員權限

### 準備解決方案
- 📝 準備好記錄錯誤訊息
- 🔍 學習查看日誌文件的方法
- 💾 確認網路連接和防火牆設定
- 📱 準備備用的 API 提供商
