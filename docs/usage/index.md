### 🔧 **練習環境**
準備一個合適的練習環境：

```bash
# 創建練習專案
mkdir opencode-practice
cd opencode-practice

# 初始化一個簡單的專案
npm init -y
git init

# 創建一些基本檔案
touch README.md src/index.js src/utils.js

# 啟動 OpenCode 開始練習
opencode
```
